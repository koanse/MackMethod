Руководство оператора

Назначение программы

Данная программа позволяет  распределить исполнителей по работам с целью минимизации общих затрат методом Мака.

 

Требования к техническим и программным средствам

1.     Компьютер с характеристиками, достаточными для нормальной работы 32-рахрядной версии Windows XP (Service Pack 2).

2.     Microsoft Framework не ниже версии 2.0.

3.     VGA-монитор, клавиатура, мышь.

 

Описание интерфейса и порядка ввода данных

Работа программы происходит в однопользовательском режиме. Необходимые входные параметры (размерность и матрицу затрат) пользователь задает в соответствующих элементах управления главного окна программы.

Вначале необходимо ввести желаемую размерность матрицы затрат. Затем в главном окне появится таблица соответствующего размера. В ячейки таблицы вводим коэффициенты сложности выполнения данным работником определенной работы (это должны быть целые неотрицательные числа).

После того, как задана размерность задачи и проинициализированы все элементы матрицы затрат, в главном меню программы требуется выбрать пункт «Вычисления», и далее «Вычислить».

Если элементы матрицы затрат заданы некорректно, вычисления не будут выполнены и появится соответствующее сообщение с рекомендациями по исправлению ошибки.

 

Представление результатов

В случае, если входные данные заданы правильно, появится диалоговое окно с результатами оптимизации и подробными промежуточными вычислениями, которые при необходимости можно скопировать в текстовый документ.
